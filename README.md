# Task1Reactjs - Web Application Project
Task1 - ReactJS 

## Technologies Used

- React
- Vite
- React Router
- Axios
- Tailwind CSS

  
### Installation



1. **Clone the repository:**

   git clone https://github.com/BahaaAbbas/Task1Reactjs.git

2. **Create folder and navigate :**

   cd my-react-app

3. **Install the packages and everything related:**

   npm install

4. **Run to Test:**
   
   npm run dev
