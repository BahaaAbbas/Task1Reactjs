### Installation

1. **Create folder and navigate :**
   
   cd my-react-app

2. **Clone the repository:**

   git clone https://github.com/BahaaAbbas/Task1Reactjs.git

3. **Install the packages and everything related:**

   npm install

4. **Run to Test:**
   
   npm run dev
